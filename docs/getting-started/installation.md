# Installation

## Prerequisites
- Python 3.9+
- Docker
- kubectl
- Helm
- (Optional) AWS CLI (for AWS deployments)
- (Optional) Heroku CLI (for Heroku deployments)

## Install SFAI SDK

```bash
pip install sfai-sdk
```

For more details, see the [main README](https://github.com/salesforce/sfai-sdk/blob/main/README.md).
